# Plot length at age

Plot the mean length (cm) at age by year and season. Plus and minus one
standard deviation is also plotted.

## Usage

``` r
plot_length_at_age(data, years = c(1931, 2022))
```

## Arguments

- data:

  a `list` containing the data that was passed to `MakeADFun`.

- years:

  the years to show on the plot.

## Value

a `ggplot2` object.
