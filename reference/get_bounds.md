# Get default parameter bounds

Get `data.frame` of default parameter bounds.

## Usage

``` r
get_bounds(obj, parameters)
```

## Arguments

- obj:

  a `list` specifying the AD object created using the `MakeADFun`
  function.

- parameters:

  a `list` specifying the AD object created using the `MakeADFun`
  function.

## Value

a `data.frame` of parameter bounds.
