# Compute weight at each length bin midpoint

Compute weight at each length bin midpoint

## Usage

``` r
get_weight_at_length(len_mid, lw_a, lw_b)
```

## Arguments

- len_mid:

  Numeric vector. Length bin midpoints (data).

- lw_a:

  Numeric. L-W scalar (data).

- lw_b:

  Numeric. L-W exponent (data).

## Value

Numeric vector of weight at each length bin.
