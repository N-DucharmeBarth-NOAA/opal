# Convert grid to an snutsfit object

Convert grid to an snutsfit object

## Usage

``` r
grid_to_snutsfit(data, parameters, grid, grid_parameters, grid_cells)
```

## Arguments

- data:

  a `list` of model runs.

- parameters:

  a `list` of model runs.

- grid:

  a `list` of model runs.

- grid_parameters:

  a `list` of model runs.

- grid_cells:

  a `list` of model runs.

## Value

a `vector` of integers.
